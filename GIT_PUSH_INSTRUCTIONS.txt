STEPS TO PUSH:
--------------

1. Initialize Git Repository (if not already done):

   cd D:\24_StepAnalyzer
   git init


2. Add Remote Repository:

   git remote add origin https://github.com/xiaojunyang0805/2024_StepAnalyzer.git


3. Remove old folder manually (if still exists):

   # Close any running processes first, then:
   rm -rf StepAnalyzer_v3_Windows


4. Stage Files (respects .gitignore, Script/ will be excluded):

   git add .


5. Review What Will Be Committed:

   git status

   # Should show:
   # - StepAnalyzer_v3/
   # - TestData/
   # - SciPaper/
   # - README.md
   # - .gitignore
   #
   # Should NOT show:
   # - Script/
   # - App/


6. Commit Changes:

   git commit -m "Release StepAnalyzer v3.0.0 - Standalone Windows Application

   Features:
   - Standalone Windows executable (no Python required)
   - Interactive GUI with 4-panel visualization
   - Batch processing with histogram and Gaussian fitting
   - Quality review dialog with interactive filtering
   - Professional Excel/CSV/JSON export
   - Sample test data and scientific documentation included

   Package size: ~254 MB
   Windows 10+ (64-bit) compatible"


7. Push to GitHub:

   git branch -M main
   git push -u origin main

   # Or if repository already exists:
   git push origin main


8. Create Release (on GitHub):

   a. Go to: https://github.com/xiaojunyang0805/2024_StepAnalyzer/releases
   b. Click "Create a new release"
   c. Tag version: v3.0.0
   d. Release title: StepAnalyzer v3.0.0
   e. Description: Copy from CHANGELOG.md
   f. Upload StepAnalyzer_v3.zip as release asset
   g. Publish release
